## Bitscale Dashboard

A simple dashboard.

### 🔧 Tech Stack

- [Next.js](https://nextjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn ui](https://ui.shadcn.com/)

### 📦 Getting Started

- Clone the repo

```bash
git clone https://github.com/aviralj02/bitscale-dashboard
cd bitscale-dashboard
```

- Install dependencies and spin up the dev server:

```bash
yarn install
yarn dev
```
